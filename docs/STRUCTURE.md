# NymyaLang Project Structure

## Directories
- **src/** - Source code including compiler (nymyac) and core libraries
- **lib/** - Library implementations 
- **docs/** - Documentation files
- **tests/** - Test files and verification programs
- **tools/** - Development and build tools
- **examples/** - Example programs
- **build_configs/** - Build configuration files
- **temp/** - Temporary files and work-in-progress

## Main Components
- **compiler (nymyac)** - Located at src/nymyac/
- **Standard libraries** - Located at src/nymyac/library/
- **Documentation** - Located at docs/

This structure provides a clear separation of concerns for development and maintenance.
